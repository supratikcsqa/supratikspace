# Funnel Analysis: Supratik Space Builder Lite

Date: 2026-03-31

## Funnel Impact

| Funnel Stage | Impacted | Expected Lift | Drop-off Risk Without Feature |
| --- | --- | --- | --- |
| Awareness | Yes | High: clearer positioning and use-case framing should improve relevance on first visit. | Visitors assume the site is an AI agent dashboard and bounce. |
| Activation | Yes | High: GitHub import plus instant subdomain preview reduces setup time and cognitive load. | Busy builders do not start because setup feels vague or manual. |
| Engagement | Yes | Medium: the dashboard becomes a reason to return when work changes. | Sites become stale and users stop caring after launch. |
| Retention | Yes | Medium: hosted project pages and connected profiles keep the site useful over time. | Builders move back to GitHub Pages, Framer, or static personal sites. |
| Revenue | Yes | High: a low-friction $10 purchase gives an immediate monetization step. | Interest never turns into a transaction because there is no clear paid offer. |

## Jobs To Be Done

### Primary Job

When I am a busy builder with projects spread across GitHub and creator tools, I want a personal site generated from my live work, so I can look credible and current without becoming a full-time website maintainer.

### Secondary Jobs

1. When I launch an open-source project, I want it to live on a clean subdomain, so I can share it like a real product.
2. When someone asks for my portfolio or resume, I want one lightweight site that already reflects my latest work.
3. When I buy a custom domain, I want guided DNS setup, so I can publish under my own name without ops friction.

### Negative Jobs

1. I do not want the product to feel like another bloated website builder.
2. I do not want to manually copy the same project data into five different fields.
3. I do not want DNS, hosting, or integrations to become a support-heavy chore.

## Prioritization Note

The Product Manager Toolkit RICE model was used with directional inputs to rank the three updates:

1. Landing page pivot plus template showcase: RICE 800
2. GitHub repo to supratik.space subdomain launcher: RICE 240

This ranking supports a "today" sequence of reposition first, activation second, monetization third.
