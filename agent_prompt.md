*   **Mandatory Beginning**: You MUST start with a visceral pattern interrupt.

## CRITICAL: THE SEAMLESS LOOP

## Execution Rules
*   **Visual Change**: Every 2.5 seconds.
*   **Tone**: Edgy, urgent, authentic. No "testament" or "harness".
