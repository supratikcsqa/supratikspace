# Viral Script & Storytelling

## Context
Use for writing video scripts or high-engagement social threads.

## Script Architecture
1. **The Visual Hook**: Start with a pattern interrupt in the first 1.5 seconds.
2. **The "Wait, What?" Loop**: Introduce a mystery or question mid-script to hold retention.
3. **The Micro-CTA**: Use a non-generic call to action (e.g., "Look at the first comment").

## Cognitive Architecture
- **Rhythm Analysis**: Ensure sentence lengths vary to prevent "AI monotone."
- **Verification**: Test if the script passes the "No Audio" test (does it work on mute?).
