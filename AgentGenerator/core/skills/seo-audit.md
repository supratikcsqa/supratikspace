# Technical SEO & Site Architecture

## Context
Use when the user asks: "Why am I not ranking?" or "Audit my site's technical health."

## Audit Protocol
1. **Crawlability**: Check for `robots.txt` blocks and canonical alignment.
2. **Architecture**: Map the site hierarchy. No page should be more than 3 clicks from the homepage.
3. **CWV Audit**: Analyze LCP, FID, and CLS metrics.

## Cognitive Architecture
- **Inversion Thinking**: "What is the single biggest technical bottleneck preventing indexing?"
- **System 2 Logic**: Before proposing a change, simulate its impact on the crawl budget.
