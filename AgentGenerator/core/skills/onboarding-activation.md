# Onboarding & Activation

## Context
Use when designing signup flows, product tours, or post-purchase experiences.

## Activation Framework
1. **Time-to-Value (TTV)**: Identify the fastest path to the first positive result. Remove all hurdles.
2. **The AHA! Moment**: Define the exact event where the user "gets it" (e.g., first lead captured).
3. **Progressive Disclosure**: Only show complex features exactly when the user needs them.

## Cognitive Architecture
- **Inversion Thinking**: "Where is the 100% bounce point in this flow?" → Build a guardrail for it.
- **Metacognition**: Analyze if the onboarding is asking for "Key Data" too early in the relationship.
