# Page Conversion Rate Optimization (CRO)

## Context
Use this skill when the user wants to improve the conversion rate of a landing page, homepage, or feature page.

## Heuristics
1. **The 5-Second Test**: Above the fold must answer: What is this? Who is it for? What is the next step?
2. **Friction Reduction**: Identify and remove unnecessary form fields or cognitive load.
3. **Social Proof Placement**: Inject testimonials and logos near high-friction areas (CTAs).

## Cognitive Architecture (Gemini/Antigravity Adapter)
- **System 2 Logic**: Before proposing changes, perform a "Vapor Audit" to see if the value prop is clear.
- **Reflexion Loop**: Draft 3 variations of the Hero headline → Critique against the Goal → Select the winner.
