# SaaS Growth Experiments

## Context
Use when brainstorming growth hacks, referral systems, or viral mechanics.

## Experiment Framework
1. **Hypothesis Generation**: "If we [Change], then [Metric] will [Result] because [Reason]."
2. **The Viral Coefficient (K)**: Design loops where 1 user brings in > 1 active user.
3. **Double-Sided Incentives**: Use the "Dropbox" model (Reward both the inviter and invitee).

## Cognitive Architecture
- **System 2 Attention**: Filter out generic "Run an A/B test" advice. Focus on non-obvious levers.
- **Tree of Thoughts**: Compare the ROI of "Paid Ads" vs "Viral Product Loop" for the target market.
